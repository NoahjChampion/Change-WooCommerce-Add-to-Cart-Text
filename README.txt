== Description ==

This plugin helps you to change the "Add To Cart" button text to anything from a simple settings page available in wp-admin → Settings → Reading


= Features =
 * Unicode Character Support
 * HTML5 Special Character Support
 * Font Awesome and other available icons
 * Emoji

= Contribute =
You can fork this plugin, add some more cool features and send me pull request here [Github](https://github.com/badsha-eee/Change-WooCommerce-Add-to-Cart-Text).

= Author =
[Sekander Badsha](http://sekander.pro) from [weDevs](https://wedevs.com)

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Frequently Asked Questions ==

= Q. How to use font awesome icons? =
A. You can put the normal html code like `<i class="fa fa-shopping-cart"></i> Buy Now` in the text box and it will work like a charm. But remember this plugin does not add the fontawesome or any other icon sets. Your plugin or theme has to load the necessary icons/fonts. Else you can add them manually.

= Q. How can I add HTML5 Special Characters/Emoji? =
A. There is no magic trick. Just copy and paste!
